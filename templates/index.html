Txt2Vid

Txt2Vid is a set of tools to help you create attractive videos from boring articles.

Whats included

Txt2Vid consists the following:

  • Upload: Choose an article and upload it.
  • Convert: Convert it to a video with high quality pictures in just one click.
  • Download: Save to your system.
  • Share: Share with the world

Installation

You can install Txt2Vid using cocoapods,

Cocoapods

use_frameworks!
pod 'Swiftline'

Carthage

github 'Swiftline/Swiftline'

Cocoapods + Rome plugin

If you want to use swiftline in a script you can use Rome cocoapod plugin. This plugin builds the framework from the pod file and place them in a Rome directory.

platform :osx, '10.10'
plugin 'cocoapods-rome'

pod 'Swiftline'
